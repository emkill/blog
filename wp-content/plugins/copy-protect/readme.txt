Copy Protect will Prevent your site content from getting copied.

== Description ==

**Copy Protect** disables the keyboard and mouse controls used to copy your site content.

== Installation ==

1. Upload `copyprotect` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it, You are done. Now your site contents are protected with **Copy Protect** Plugin.

== Frequently Asked Questions ==

= How will it protect the content?  =

Copy controls such as ctrl+s,ctrl+a,ctrl+c,ctrl+u and mouse right click, mouse select are disabled when the plugin is active. By this your content is protected from getting copied.

= Does it need initial setup? =
No, the plugin doesn't need any initial setup. Copy Protect will work once the plugin is activated with out any settings.
